# Modul Pembelajaran Kali Linux untuk Keamanan Informasi
### SMKN 1 Punggelan
#### Jurusan Sistem Informasi Jaringan dan Aplikasi (SIJA)

## Tentang Modul
Modul pembelajaran ini dirancang khusus untuk siswa SMK Jurusan Sistem Informasi Jaringan dan Aplikasi (SIJA) SMKN 1 Punggelan. Materi disusun sesuai dengan:
- Kurikulum SMK terbaru
- Kebutuhan industri di bidang keamanan informasi
- Standar kompetensi lulusan SMK
- Perkembangan teknologi terkini

### Tujuan Pembelajaran Global
Setelah menyelesaikan seluruh modul ini, siswa SMKN 1 Punggelan jurusan SIJA diharapkan mampu:
1. Memahami dan menerapkan prinsip keamanan informasi
2. Mengoperasikan sistem operasi Kali Linux
3. Menggunakan tools security testing
4. Melakukan analisis keamanan jaringan
5. Membuat dokumentasi pengujian keamanan

### Prasyarat
Siswa diharapkan telah memiliki pengetahuan dasar tentang:
1. Sistem Operasi Linux
2. Jaringan Komputer
3. Pemrograman Dasar
4. Bahasa Inggris Teknis

## Detail Modul Pembelajaran

### Modul 1: Dasar Keamanan Informasi dan Pengenalan Kali Linux
**Tujuan Pembelajaran:**
- Memahami konsep dasar keamanan informasi dan pentingnya dalam dunia digital
- Menguasai prinsip CIA (Confidentiality, Integrity, Availability)
- Mampu menginstal dan mengkonfigurasi Kali Linux

**Materi Utama:**
1. Pengertian Keamanan Informasi
   - Definisi dan ruang lingkup
   - Ancaman dan risiko keamanan
   - Dampak pelanggaran keamanan

2. Prinsip CIA
   - Confidentiality: menjaga kerahasiaan data
   - Integrity: memastikan keutuhan data
   - Availability: menjamin ketersediaan layanan

3. Pengenalan Kali Linux
   - Sejarah dan evolusi Kali Linux
   - Fitur dan tools utama
   - Perbandingan dengan distro Linux lain

4. Instalasi dan Konfigurasi
   - Metode instalasi (VM, dual boot, live USB)
   - Konfigurasi dasar sistem
   - Manajemen paket dan repositori

### Modul 2: Fundamental Ethical Hacking dengan Kali Linux
**Tujuan Pembelajaran:**
- Memahami konsep dan metodologi ethical hacking
- Menguasai konfigurasi lanjutan Kali Linux
- Mampu menerapkan prinsip keamanan dasar

**Materi Utama:**
1. Ethical Hacking
   - Definisi dan ruang lingkup
   - Metodologi pengujian
   - Aspek legal dan etika

2. Konfigurasi Lanjutan
   - Setup lingkungan virtualisasi
   - Konfigurasi jaringan
   - Hardening sistem

3. Tools Dasar
   - Terminal dan command line
   - Scripting dasar
   - Tool management

### Modul 3: Information Gathering dan Footprinting
**Tujuan Pembelajaran:**
- Menguasai teknik pengumpulan informasi

2. Active Information Gathering

   - Service enumeration
   - OS fingerprinting

3. Social Engineering
   - Teknik dasar

- Kali Linux (latest version)
- VMware Workstation/VirtualBox
